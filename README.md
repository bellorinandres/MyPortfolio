# Portfolio de Andres Bellorin

Portfolio pessoal criado com Astro para apresentar experiência profissional, habilidades, idiomas e projetos.

## Stack

- Astro 5
- TypeScript
- HTML e CSS
- hotkeypad para paleta de comandos

## Como executar

```sh
npm install
npm run dev
```

## Comandos

```sh
npm run dev
npm run build
npm run preview
```

## Conteúdo

As informações do currículo ficam em `src/cv.json`. Para atualizar nome, resumo, experiência, projetos, habilidades ou idiomas, edite esse arquivo.

## Estrutura Principal

- `src/pages/index.astro`: página inicial.
- `src/Layouts/Layouts.astro`: HTML base, metadados e estilos globais.
- `src/components/sections`: seções do portfolio.
- `public`: imagens e arquivos públicos.
